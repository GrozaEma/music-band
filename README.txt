READ ME

Start the server app:

    - in folder "SimpleChat" run "node chat-server"

-------------------------------------------------------------------------------------------

Start the web app:

    You need http-server to serve static files, in order to use the web app.
    Run: "npm install http-server -g"
    
    To start the server: 
        - go to "final_app_client" folder and run "http-server"

-------------------------------------------------------------------------------------------

Then access from Firefox (Chrome won't work) 
    
    localhost:8080/#/music-band
